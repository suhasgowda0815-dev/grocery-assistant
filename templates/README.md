# Grocery Inventory Assistant

AI-powered grocery inventory management system using AWS Bedrock and DynamoDB.

## Features
- Real-time inventory tracking
- Recipe suggestions based on available ingredients
- AI-powered chat assistant
- AWS App Runner deployment

## Setup
1. Create DynamoDB tables (GroceryInventory and GroceryRecipes)
2. Configure AWS credentials with appropriate permissions
3. Deploy to AWS App Runner

## Local Development
```bash
pip install -r requirements.txt
python app.py
```

Visit http://localhost:8080
